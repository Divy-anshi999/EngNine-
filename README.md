
 Engnine – Educational Platform Website

Engnine is a responsive multi-page educational platform website created to showcase an interactive learning environment. The project presents training programs, personality development content and learning methodologies using a clean and user-friendly interface.

The website demonstrates front-end development fundamentals such as structured layout design, navigation handling and responsive UI components.

---




 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript** – Basic interactivity

---

## 🎯 Purpose of the Project

The purpose of this project is to practice and demonstrate core front-end development skills including:

* Building a structured multi-page website
* Designing a responsive layout
* Creating reusable UI sections
* Improving user interface readability

🚀 How to Run Locally

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`
